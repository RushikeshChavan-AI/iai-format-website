The idea

A document package that stays editable after AI creates it.

IAI packages are designed to hold structured source content, preview files, styles, media assets, component metadata, and AI history together. The goal is to avoid the usual dead end where AI output becomes a flat PDF or static page too early.

  • Editable `.iai` package for authoring
  • Portable `.iaip` package for sharing previews
  • Native desktop editor for Windows builds
